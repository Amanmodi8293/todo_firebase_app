# ✅ Flutter Firebase Todo App

This is a simple Todo App built using **Flutter**, **Firebase Authentication**, and **Cloud Firestore**. It allows users to:

- 🔐 Register & Login
- ✅ Add Todo items
- 📝 View Todos
- ✏️ Edit Todos
- ❌ Delete Todos

---

## 🔧 Technologies Used

- Flutter (Web & Mobile)
- Firebase Auth
- Cloud Firestore
- Dart
